# Stock-Prediction-App
A Stock Prediction app using a Machine Leraning Model


#📌Project Title:

Stock Price Prediction 


#📝Project Description:

This project predicts and visualizes stock price trends using machine learning and deep learning techniques. Users can explore real-time stock data, analyze historical trends, and forecast prices for the next 7 days through an interactive dashboard.

#📂 Project Structure / Features:

*Real-time stock price input for any ticker symbol

*Interactive visualizations with candlestick charts

*Historical data analysis and trend detection

*7-day future stock price forecasting using ML/DL models

*User-friendly dashboard interface for dynamic exploration


#📊 Tools & Technologies:

*Python (Pandas, NumPy, Matplotlib, Plotly, Dash/Streamlit/Flask)

*Machine Learning (Linear Regression, Random Forest, LSTM)

*Deep Learning (TensorFlow/Keras)

8Front-end Dashboard ( Streamlit)


#Data Source: 

Yahoo Finance API
